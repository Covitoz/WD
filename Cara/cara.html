Cara Install HFW / HEN 4.89.1 PRO

- STEP 1 -


Pastikan Flasdisk / Harddisk berformat FAT32 (WAJIB)
Ekstrak file HFW 4.89.1 PRO

Urutannya yaitu (PS3 > UPDATE > PS3UPDAT.PUP)
